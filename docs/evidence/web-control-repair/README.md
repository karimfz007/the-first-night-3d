# Web control repair evidence

The PNG files in this directory are generated by
`tests/browser/web_control_repair.spec.mjs` from the exported Web build. They
cover the desktop initial/focus surface, desktop settings, Android-landscape
controls, fire placement preview, and successfully placed fire.

The assertions also exercise 740×360, 915×412, 1024×600, and portrait
orientation sizes, a background/resume transition, simultaneous movement/look
touches, pointer-lock capture/release, hotbar selection, placement cancel and
confirmation, and campfire ignition. Run with `npm ci && npm run test:browser`
after exporting Web to `builds/web/index.html`.
