# Project Codex

## Terms

- **Brain:** deterministic scene-free world rules and serializable state.
- **Body:** Godot scene, input, presentation, feedback, and platform adapters.
- **First night:** the dusk-to-night survival loop in slice 0.1.
- **Morning report:** factual offline reconciliation summary shown on return.
- **Sanctuary:** readable safety created by fire heat plus shelter protection.
- **Stable ID:** deterministic identifier persisted across save/reload.
- **Knowledge hook:** recipe gate reserved for embodied progression.
- **Protection:** bounded contribution from complete nearby shelter pieces.

## Canonical identifiers

Items: `driftwood`, `deadfall`, `stone`, `plant_fiber`, `stone_tool`,
`building_plan`, `campfire_kit`.

Recipes: `primitive_stone_tool`, `campfire`, `building_plan`.

Build pieces: `foundation`, `wall`, `doorway`, `door`, `roof`, `campfire`.

## Tone vocabulary

Use terse, physical language: cold, exposed, sheltered, warm, dry wood, stone,
embers, horizon. Mystery language should describe observable anomalies without
explaining them.
